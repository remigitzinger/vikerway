# Vikerway

**Where complexity becomes a decision.**

A minimalist website for Vikerway, a boutique advisory firm operating across high-stakes financial and strategic matters.

## Design Philosophy

The site embodies restraint, authority, and strategic absence:

- **Minimal content**: What is not said is as important as what is
- **Monumental typography**: Serif headings (Cormorant Garamond) paired with clean sans-serif body text
- **Color palette**: Absolute black, pure white, and a touch of gold
- **Generous whitespace**: Luxury through emptiness

## Pages

1. **Homepage** — Single statement with minimal CTA
2. **About** — Identity statement, not biography
3. **Scope** — Deliberately elliptical description of engagement range
4. **Contact** — Understated form with visible email

## Domain

Vikerway.eu — registered via EU-based registrar to reinforce European sovereignty.

## Deployment

Hosted on Infomaniak or OVHcloud for full EU data sovereignty.
